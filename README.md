# System Cleaner

System Cleaner is a lightweight Linux cleanup utility.

It can clean:
- User cache
- Temporary directories
- Journal logs
- Package manager caches (apt, dnf, pacman, zypper)

## Build

```bash
make
```

## Run

```bash
./system-cleaner
```

Run as root for full system cleanup.
Run as a normal user for limited cleanup.
